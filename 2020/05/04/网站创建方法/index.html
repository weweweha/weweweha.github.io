网站创建方法

  一开始创建这个博客是想用腾讯云的域名和云服务器组合，可是用wordpress+宝塔傻瓜式创建好博客之后发现国内服务器是一定要备案的，用了还没半小时就挂了。
  这怎么办呢，总不能再去组个香港服务器吧，贫穷的我开始寻找不需要服务器的博客创建方法，搜了一下其实还不少，原来还可以把博客挂在GITHUB或者国内的码云上面，可以直接用github.io作为域名来访问，如果已经有个人域名了，还可已解析到github上面，这样即使在国内也能快速访问这个博客了。
  网上有很多人都做过教程，我在这里就当做复习了。
  首先需要两个软件nodejs  GIT
  安装好之后再Powershell中用node -v检查时候安装是否完成
  如图所示就是安装完成了，git安装时候成功只要在桌面邮件看有没有GITBASH和GITGUI就可以了
  之后在C:/USERS/（你的用户名）中创建一个BLOG文件夹，直接POWERSHELL：mkdir blog来创建，建议使用命令行创建，之后的操作基本都是在POWERSHELL中完成。
  之后安装CNPM，由于国外镜像较慢所以用淘宝镜像来安装（说实话这一步我并不太理解，但确实比直接下载快）,直接下载：npm -g install cnpm,淘宝镜像下载：npm -g install cnpm –registry=https://registry.npm.taobao.org
  完成安装CNPM之后就可以安装HEXO框架了，命令行输入：cnpm install -g hexo-cli,如果这一步CNPM报错，那可能是没有吧CNPM设置为全局变量。
  在命令行中进入刚刚创建的BLOG文件夹，输入hexo init（WINDOWS系统），进行HEXO框架的安装，这里可能会比较慢，挂个LADDER可能会快一点。
  到这一步HEXO就算是安装完成了，命令行中输入hexo s来开启博客，命令行中会给出4000端口，复制到浏览器中就能打开。
  但是博客肯定不是就放在自己电脑上看的，所以后面就要把HEXO来推到GIT上面。
  这一步中首先要去GITHUB中创建账号和仓库，然后在BLOG的CONFIG文件中的最后把TYPE改成GIT，再加上两行参数，分别是repo:你的仓库地址，BRANCH：MASTER，注意这里两个冒号之后都要加上一个空格，不然识别不了，然后就可以上传了，在BLOG目录下面输入hexo d，博客就推送到你的GITHUB仓库中去了，之后使用你建仓库的名称，比如说：weweweha.github.io就可以访问你的博客了，这里有一点前面忘记忘记说了，存放你博客的仓库名称必须是以github.io结尾，不然不能访问。
  如果你有个人域名的话，可以直接把域名解析到你GITHUB的域名中去，IP地址用命令行PING一下就可以得到了。
  如何写博客和如何改变博客的主题，各位可以在网上自行搜索。
