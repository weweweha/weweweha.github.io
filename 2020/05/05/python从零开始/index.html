PYTHON从零开始

从零开始可能有些夸张，但是确实基本功不行，"从零开始复习"可能更好，最近在学习PYTHON爬虫的时候发现不会CSS和HTML是不能很好的了解爬虫的，所以在PYTHON学习的时候我也会尽量地去学习HTML和CSS的知识，我觉得学习一样新东西，最好的方法就是放低自己的姿态，什么都不懂才能学更多。

​ 参考资料：菜鸟教程][https://www.runoob.com]，PYTHON编程从入门到实践（Eric Matthes）

1.变量与数据类型

​ PYTHON有六个数据类型

数据类型 是否可以改变
数字（number） FALSE
字符串（string） FALSE
列表（list） TRUE
元祖（tuple） FALSE
集合（set） TRUE
字典（dictionary） TRUE

1.1·数值运算

a=2,b=4

运算方法 结果
+(加法) 6
-（减法） -2
*（乘法） 8
**（乘方） 16
/（除法的一个浮点数） 0.5
//（除法，的一个整数，向下取整） 0
%（取余） 2
